# 🎥 튜브렌즈(TubeLens) 핵심 기능 정리

## 🧠 1. 유튜브 분석 강화
- 채널 성장 기반 분석
- 경쟁 채널 비교 및 인사이트 제공
- 트렌딩 콘텐츠 및 급상승 영상 탐색

## 🔍 2. 주요 분석 기능
- **조회수 / 구독자 대비 성과 분석**  
  → 숨은 효자 영상·떡상 영상 찾기
- **키워드 분석 & 자막 분석**  
  → 검색·노출·주제 최적화
- **댓글 감정·반응 분석**  
  → 시청자 반응·참여도 파악
- **썸네일 분석**  
  → CTR 높은 이미지 요소 파악

## 🤖 3. AI 기능
- **AI 타이틀 메이커** → 최적 제목 자동 생성
- **영상 분석 요약** → 영상 핵심 아이디어 자동 요약
- **AI 스크립트 생성** → 콘텐츠 대본 자동 생성
- **AI 트렌드/아이디어 추천** → 채널 맞춤형 주제 추천

## 💾 4. 다운로드 기능
- 영상 **MP4 다운로드**
- 오디오 **MP3 추출**
- TikTok/Instagram/Reels/도우인 등 멀티 플랫폼 지원

## 🧩 5. 주요 활용 포인트
- 경쟁 채널 역추적 → 성장 전략 복제
- 바이럴 영상 요소 분석 → 내 콘텐츠에 적용
- 아이디어 → 제목 → 대본 → 썸네일 → 업로드까지 워크플로우 단축

## ⭐ 요약
튜브렌즈는  
**“유튜브 떡상 영상 탐색 + AI 전략 + 다운로드 툴”**  
올인원 플랫폼입니다.

> 한마디 요약: 성장하는 채널들의 비밀 레시피를 빠르게 벤치마킹하는 도구
